# BLOGORA 📝

A full-stack blogging platform built with the **MERN Stack**, focusing heavily on **backend development, authentication, authorization, routing, middleware architecture, and database management**.

## 🚀 Features

* User registration and login with **Passport.js**
* Secure **session-based authentication**
* Create and manage user profiles
* Create, edit, delete, and view blog posts
* Add and delete comments
* Image uploads using **Multer** and **Cloudinary**
* Protected routes and ownership-based authorization
* Error handling with **express-async-handler**
* MVC architecture with modular code structure

## ⚙️ Installation

```bash
git clone <repository-url>
cd blogora
npm install
npm start
```

Create a `.env` file and add your MongoDB, Cloudinary, and Session credentials.
